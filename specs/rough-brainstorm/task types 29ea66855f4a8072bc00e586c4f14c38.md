# Task Types & Scoring System Guide

## How to Track Different Actions and Calculate Category Scores

---

## **PART 1: THE 5 TYPES OF ACTIONS**

Every action in your life tracking system falls into one of these types:

```
┌─────────────────────────────────────────────────────────────────────┐
│                    5 TYPES OF TRACKABLE ACTIONS                     │
└─────────────────────────────────────────────────────────────────────┘

1. 🔄 RECURRING HABITS
   → Things you do regularly (daily/weekly)
   → Examples: Meditate 10min, Gym 3x/week, Call parents 2x/week

2. ✅ ONE-TIME TASKS
   → Specific things to complete once
   → Examples: Book dentist appointment, Fix leaky tap, Buy gym shoes

3. 🔬 EXPERIMENTS (Time-bound projects)
   → Test something for a defined period
   → Examples: Try meal prep for 4 weeks, Go to gym daily for 30 days

4. 🎯 ONGOING PROJECTS
   → Multi-step efforts with milestones
   → Examples: Build side project, Learn Spanish, YouTube channel

5. 📝 REFLECTIONS/JOURNALING
   → Thought work, values alignment, self-awareness
   → Examples: Weekly reflection, Monthly values check-in, Gratitude practice

```

## **PART 2: DATABASE STRUCTURE FOR EACH TYPE**

### **Option A: Single "Actions" Database (Recommended)**

One master database with a "Type" property to categorize everything.

**Database Name:** "Life Actions Tracker"

**Properties:**

```
┌──────────────────────────────────────────────────────────────────┐
│ PROPERTY NAME       │ TYPE         │ OPTIONS/DETAILS             │
├──────────────────────────────────────────────────────────────────┤
│ Action Name         │ Title        │ The thing you're tracking   │
│ Category            │ Select       │ 6 categories (Physical, etc)│
│ Type                │ Select       │ 5 types (see below)         │
│ Status              │ Select       │ Active/Paused/Complete/Drop │
│ Date                │ Date         │ For tracking when           │
│ Completed?          │ Checkbox     │ For daily check-offs        │
│ Streak              │ Number       │ Current streak count        │
│ Frequency           │ Select       │ Daily/3x week/Weekly/etc    │
│ Points              │ Number       │ For scoring (1-5)           │
│ Progress            │ Number       │ % complete (for projects)   │
│ Start Date          │ Date         │ When you started            │
│ End Date            │ Date         │ For experiments/projects    │
│ Notes               │ Text         │ Any additional info         │
│ Related To          │ Relation     │ Link to project/experiment  │
└──────────────────────────────────────────────────────────────────┘

TYPE OPTIONS:
- 🔄 Recurring Habit
- ✅ One-Time Task
- 🔬 Experiment
- 🎯 Project
- 📝 Reflection

```

### **How Each Type Uses the Database**

### **🔄 TYPE 1: RECURRING HABITS**

**Example Entry:**

```
Action Name: Morning Meditation
Category: Mental Mastery
Type: 🔄 Recurring Habit
Status: Active
Frequency: Daily
Points: 3 (medium importance)
Streak: 8 days
Date: [Today's date] (auto-updates)
Completed?: ☐ (check daily)

```

**How it works:**

- Create ONE entry per habit
- Each day, check the "Completed?" box
- Streak counter updates (manual or formula)
- In dashboard view, filter to show "Today" habits only

**Dashboard View Settings:**

- Filter: Type = "Recurring Habit" AND Status = "Active"
- Show: Action Name, Category, Completed?, Streak
- Sort: Category, then Points (highest first)

---

### **✅ TYPE 2: ONE-TIME TASKS**

**Example Entry:**

```
Action Name: Book annual health checkup
Category: Physical Foundation
Type: ✅ One-Time Task
Status: Active
Due Date: Oct 30, 2025
Completed?: ☐
Points: 2 (nice to have)

```

**How it works:**

- Create entry when you think of task
- Check off when complete
- Status changes to "Complete" (manual or automation)
- Archived or moved to "Done" view

**Dashboard View Settings:**

- Filter: Type = "One-Time Task" AND Status = "Active"
- Show: Action Name, Category, Due Date, Completed?
- Sort: Due Date (urgent first)

---

### **🔬 TYPE 3: EXPERIMENTS**

**Example Entry:**

```
Action Name: 30-Day Gym Streak Experiment
Category: Physical Foundation
Type: 🔬 Experiment
Status: Active
Start Date: Oct 1, 2025
End Date: Oct 30, 2025
Progress: 75% (23/30 days completed)
Points: 5 (high priority this month)
Success Criteria: Go to gym 30 days straight
Current Status: Day 23, feeling strong!

```

**How it works:**

- Create entry at start of experiment
- Track progress in notes or separate related tasks
- Update Progress % as you go
- At end date, mark Complete and write learnings

**Related Habits:**

- You can create daily recurring habits linked to this experiment
- Example: "Gym session" habit with "Related To: 30-Day Gym Experiment"

**Dashboard View Settings:**

- Filter: Type = "Experiment" AND Status = "Active"
- Show: Action Name, Progress %, Days Remaining, Status
- Could show as progress bars

### **📝 TYPE 5: REFLECTIONS/JOURNALING**

**Option A: Keep in Journal Database** (Recommended)

- Don't track these as "actions"
- Keep in separate Journal database
- Link journal entries to categories via tags

**Option B: Track as Recurring Habits**

```
Action Name: Weekly Reflection
Category: Existential Design
Type: 📝 Reflection (or 🔄 Recurring Habit)
Frequency: Weekly (Sundays)
Points: 3
Status: Active
Completed?: ☐

```

**How it works:**

- If it's a habit (do reflection regularly), track as recurring
- The actual reflection content goes in Journal database
- Check off the habit when you complete your journaling

## **PART 3: SCORING METHODOLOGY**

### **THE SCORING FORMULA**

Each category gets a score of 1-10 at the end of each month based on:

```
CATEGORY SCORE = Weighted Average of 3 Factors

┌────────────────────────────────────────────────────────────────┐
│ FACTOR 1: HABIT CONSISTENCY (40% weight)                       │
│ ────────────────────────────────────────────────────────────   │
│ How consistently did you do your recurring habits?             │
│                                                                 │
│ Formula:                                                        │
│ (Total habits completed / Total habits tracked) × 10           │
│                                                                 │
│ Example:                                                        │
│ Physical Foundation has 4 daily habits                          │
│ Month has 30 days = 120 possible check-offs                    │
│ You completed 96 check-offs                                    │
│ Score: (96/120) × 10 = 8.0                                     │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│ FACTOR 2: PROGRESS ON GOALS (30% weight)                       │
│ ────────────────────────────────────────────────────────────   │
│ Did you make progress on projects/experiments?                 │
│                                                                 │
│ Formula:                                                        │
│ Average progress % across all active projects × 10             │
│                                                                 │
│ Example:                                                        │
│ Professional Excellence has 2 active projects:                  │
│ - Side project: 60% complete                                   │
│ - Online course: 40% complete                                  │
│ Average: 50%                                                    │
│ Score: 50% × 10 = 5.0                                          │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│ FACTOR 3: SUBJECTIVE FEEL (30% weight)                         │
│ ────────────────────────────────────────────────────────────   │
│ How do you FEEL about this area of life?                       │
│                                                                 │
│ Rate 1-10 based on:                                             │
│ - Are you satisfied?                                            │
│ - Do you feel you're growing?                                   │
│ - Is this area bringing you joy/peace?                          │
│ - Are you proud of your efforts?                                │
│                                                                 │
│ Example:                                                        │
│ Physical Foundation:                                            │
│ "I feel great! I'm getting stronger, sleeping better,          │
│  and have more energy. I'd rate this 9/10."                    │
│ Score: 9.0                                                      │
└────────────────────────────────────────────────────────────────┘

FINAL CATEGORY SCORE:
= (Factor 1 × 0.4) + (Factor 2 × 0.3) + (Factor 3 × 0.3)

Example for Physical Foundation:
= (8.0 × 0.4) + (5.0 × 0.3) + (9.0 × 0.3)
= 3.2 + 1.5 + 2.7
= 7.4/10 🟢

```

---

### **SIMPLIFIED SCORING (If Math Feels Like Too Much)**

Just use **subjective intuitive scoring**:

```
At end of month, ask yourself for each category:

"On a scale of 1-10, how did I do this month?"

Consider:
├─ Did I show up consistently? (habits)
├─ Did I make progress? (projects/tasks)
├─ Do I feel good about this area? (feelings)
└─ Am I trending in the right direction? (trajectory)

Trust your gut. Write 2-3 sentences explaining the score.

Example:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💪 Physical Foundation: 7/10 🟢

Went to gym 12 times (goal was 12), meal prepped 3/4 Sundays,
but sleep was inconsistent (7.2hrs average vs 8hr goal).
Overall trending well, body feels stronger. Need to prioritize
sleep next month.
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```

**This is recommended!** Don't overcomplicate with formulas unless you enjoy that.

## **PART 4: HOW DIFFERENT TYPES AFFECT SCORING**

### **Scoring Impact by Type:**

```
┌────────────────────────────────────────────────────────────────┐
│ TYPE               │ HOW IT AFFECTS SCORE                       │
├────────────────────────────────────────────────────────────────┤
│ 🔄 Recurring       │ HIGH IMPACT                                │
│    Habits          │ • Tracked daily/weekly                     │
│                    │ • Consistency = biggest score factor       │
│                    │ • 5-7 habits per category max              │
├────────────────────────────────────────────────────────────────┤
│ ✅ One-Time        │ MEDIUM IMPACT                              │
│    Tasks           │ • Completing them feels good               │
│                    │ • Shows you're making things happen        │
│                    │ • Count toward "progress" factor           │
├────────────────────────────────────────────────────────────────┤
│ 🔬 Experiments     │ MEDIUM-HIGH IMPACT                         │
│                    │ • Usually monthly focus                    │
│                    │ • Success/failure informs next steps       │
│                    │ • Major boost if successful                │
├────────────────────────────────────────────────────────────────┤
│ 🎯 Projects        │ MEDIUM IMPACT                              │
│                    │ • Progress over perfection                 │
│                    │ • Measured by milestones hit               │
│                    │ • Long-term trajectory matters             │
├────────────────────────────────────────────────────────────────┤
│ 📝 Reflections     │ INDIRECT IMPACT                            │
│                    │ • Helps you understand other areas         │
│                    │ • Informs your "subjective feel" score     │
│                    │ • Quality > quantity                       │
└────────────────────────────────────────────────────────────────┘

```

## **THE SIMPLE VERSION (If This Feels Overwhelming)**

Forget all the complexity. Just do this:

```
1. One database: "Life Tracker"

2. Three columns:
   - Thing I'm doing
   - Category (6 options)
   - Done? (checkbox)

4. Monthly: Look at it all and rate each category 1-10 based on gut feel

5. Pick lowest score → Make that next month's focus
```
