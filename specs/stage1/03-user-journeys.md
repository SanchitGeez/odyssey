# Odyssey - User Journeys & Stories

## Primary User Persona

**Name**: Arjun
**Age**: 24
**Location**: Bangalore
**Occupation**: Software Developer

**Pain Points**:
- Using 10+ apps - data scattered everywhere
- Rigid habit trackers don't fit real life (not everything is daily)
- Feels guilty when breaking streaks
- Can't see the big picture of life
- Tried many systems, all abandoned within 2 weeks

**Goals**:
- Get fitter and healthier
- Build better work-life balance
- Stay connected with family
- Understand what actually matters
- Track without feeling overwhelmed

## User Journey 10: Streak Breaks vs Task Skips (Understanding)

### Key Moments

**Scenario 1: User Checks In, Skips All Tasks**
```
Check-in complete!

42-day streak continues ✓

You responded to all 5 cards.
(Even if you skipped tasks, showing up = streak!)

Task completions:
✗ All tasks skipped today

That's okay. Tomorrow is a new day.
```

**Streak maintained** because user opened app and responded.

**Scenario 2: User Doesn't Open App**
- Streak breaks
- On return: "Your X-day streak ended"
- No guilt, just information

**Scenario 3: Flexible Task Past Due**
```
⚠️ Overdue Task

"File taxes" was due to be completed
sometime in March. It's now April 1.

This won't break your check-in streak,
but it's marked as overdue.

[Mark as done] [Extend deadline] [Archive]
```

## Design Principles Demonstrated

1. **Flexible Task Types** - Three types match how life actually works (recurring, one-time, projects)
2. **Speed First** - 60-90 second daily commitment for check-ins
3. **Non-Judgmental** - Skips don't break streaks, showing up is what matters
4. **Holistic View** - 6 categories prevent tunnel vision on single area
5. **Smart Suggestions** - Pre-populated task library reduces decision fatigue
6. **Separate Long-Term Tracking** - Projects (both build & experiments) don't clutter daily flow
7. **Activity Timeline** - Complete history for pattern recognition
8. **Project Flexibility** - Supports both multi-step builds and behavior experiments

## Conclusion

Arjun's journey shows Odyssey succeeding where other trackers fail:

**Not rigid** - Adapts to how life actually works
**Not punitive** - Showing up matters more than perfection
**Not fragmented** - One place for all life dimensions
**Not time-consuming** - 60-90 seconds daily

The result: Sustainable self-improvement through awareness, not guilt.
